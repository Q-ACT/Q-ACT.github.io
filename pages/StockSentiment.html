Stock Sentiment Analyzer

My final project for CSCI 331.

About this project:

This is a group project I helped create in my CSCI 331 class. It features an web front end using Flask, and backend written in python. The app provides an overview of the performance of stocks entered in the search bar and allows users to save stock symbols in order for the program to send email alerts to them when a notable change occurs.

Language(s) used:

Python, HTML, javascript, CSS

Authors:
