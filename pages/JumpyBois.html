JumpyBois

My Very First Project.

About this project:

'JumpyBois' is the first project I put on my github profile and one of the first projects I created. This was a simple mobile platformer game where players ran along a scrolling floor and could tap the screen to jump. Creating this game taught me a lot of how programming and app development works and I worked on it for several months. I no longer have the game installed on my phone so the best showcase I have of this project are the sprites I made for the game.
