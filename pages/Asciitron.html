Asciitron

A side project I made to send goofy images to my friends.

About this project:

One of my earlier projects. I created this in order to produce images made out of ascii characters so I could send my friends pictures through text. This app prompts a user to enter an image through the command console along with several settings to produce the message. The image is then put into a text file with pixels represented as ascii characters for copy and pasting.
