DinoLearn

A side project I made to play with machine learning.

About this project:

I made this in order to learn the basics of machine learning. This has an AI play a game similar to the 'dinosaur game' which appears as an easter egg on google chrome when a user fails to connect to the internet. This project uses the tensorflow library in order to use neural networks along with a basic algorithm which uses natural selection to produce the best playing dinosaurs.
