Surrounded

My final project for CSCI 265.

About this project:

This was a group project created for CSCI 265. We used Unity to create a rougelike game set in space. The player progresses through rounds and is able to unlock upgrades for their ship while enemies progressively get more difficult and more varieties appear. Plans to add boss fights and meta progression were in place but became stretch goals for this project

Language(s) used:

C#

Authors:
